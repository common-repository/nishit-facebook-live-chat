== Description ==
This plugin is developed for easily start facebook live chat for page.
== Installation ==
Simpely install from wordpres.org and active plugin after that navigate to admin menu "Facebook Live Chat" and insert your page id.
You can find page from your page's about section.
After that white list your web url in facebook page for that go to page settings >> Messanger Platform you will find Whitelisted Domain option add your domain there.
